# LIST OF GIT COMMANDS I USED

1> git init {IT INITIALIZES THE LOCAL REPO AS GIT REPO}\
#IF YOU WANT TO TURN A DIRECTORY INTO GIT DIRECTORY WHERE YOU CAN ADD OR COMMIT A FILE
2> git config --global user.name <enter_name> { USED TO CHANGE USER NAME}
TO SET USERNAME 
3> git config --global user.email <enter_emailID> { USED TO CHANGE USER EMAIL}
TO SET USER EMAIL_ID
