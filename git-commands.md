# LIST OF GIT COMMANDS I USED

1> git init {IT INITIALIZES THE LOCAL REPO AS GIT REPO}\
#IF YOU WANT TO TURN A DIRECTORY INTO GIT DIRECTORY WHERE YOU CAN ADD OR COMMIT A FILE

2> git config --global user.name <enter_name> { USED TO CHANGE USER NAME}
TO SET USERNAME
 
3> git config --global user.email <enter_emailID> { USED TO CHANGE USER EMAIL}
TO SET USER EMAIL-ID

4> git add . { IT MOVES YOUR FILE TO STAGING AREA }

5> git commit -m "<Message>" { COMMITS YOUR FILE(S) 

6> git log {shows commit history }

7> git branch (To see which all branches the user have and header shows in which branch the user is currently working at)

8> git checkout -b <new-branch-name> (makes a new branch and takes you there at the same time)

9> git switch <branch-name> ( takes you to an existing branch)

10>  git branch -d feature-2 ( To delete a local branch)
